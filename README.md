# RBGIF Teen Stories

RBGIF Teen Stories is a Progressive Web App (PWA) based Android application that presents inspiring, light-hearted, and meaningful teenage stories.

## ✨ Feature
- The application size is very light
- Fast access & data saving
- Elegant design (night blue theme)
- Can be used like an Android app

## 📱 Download
Download APK via official page or GitHub Releases

## 🎯 Objective
Providing a modern and relevant story reading experience for Indonesian teenagers.

## 🏷️ Tagline
Cerita yang Tumbuh Bersamamu (Indonesian) and Stories That Grow With You (English)
